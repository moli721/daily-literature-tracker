# Paper Scope (Editable)

Keep papers that are clearly related to **AI Agent / Agentic Systems**.

Include (any strong match is enough):
- Agent planning / reasoning / tool use / memory / orchestration
- Multi-agent collaboration / coordination / communication
- Agent safety / evaluation / reliability / robustness / security
- Agent framework, benchmark, or system design
- LLM-based autonomous workflows or agent applications

Exclude:
- Purely unrelated topics without agentic methods
- Generic ML papers with no clear agent/system relevance

Decision rule:
- If relevant to the scope above, answer **Yes**.
- Otherwise answer **No**.
